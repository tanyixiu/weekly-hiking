# 🏔️ 成都周边·每周一山

一个专为成都家庭设计的周末爬山抽签小游戏，轻松决定本周去哪座山探险！

## 🌐 在线体验

**👉 立即访问：[https://tanyixiu.github.io/weekly-hiking/](https://tanyixiu.github.io/weekly-hiking/)**

## ✨ 项目亮点

- 🎲 **随机抽签** - Fisher-Yates 洗牌算法，完全公平
- 🎨 **卡通童趣** - 明亮色彩、老虎机摇晃效果，适合全家使用
- 📊 **详细信息** - 成都周边山峰数据，包含距离、难度、风景、特产等
- 💾 **历史记录** - 确认后保存到"最终决定"
- 🔄 **可重抽** - 不满意随时再来一次

## 🚀 快速开始

```bash
cd client
npm install
npm run dev
```

访问 http://localhost:5173 开始体验
