# 🏔️ 成都周边·每周一山

一个专为成都家庭设计的周末爬山抽签小游戏，轻松决定本周去哪座山探险！

## 🌟 项目特色

- 🎲 **随机抽签** - 点击开始，让命运决定本周的目的地
- 🎨 **卡通童趣** - 色彩鲜艳、动画生动，孩子也能轻松操作
- 📊 **详细信息** - 展示距离、车程、难度、风景、特产等完整信息
- 💾 **历史记录** - 保存"最终决定"，避免重复选择
- 🔄 **可重抽** - 不满意可以再抽一次，直到找到心仪的山

## 📝 数据说明

`src/assets/mountains.csv` 包含 84 座成都周边的山，包括：

- **热门目的地**: 紫霞山、狮子宝、琉璃坝、云华山等
- **难度分级**: ★（简单）到 ★★★★★（困难）
- **覆盖区域**: 龙泉驿、大邑、彭州、崇州、都江堰等
- **详细信息**: 距离、车程、徒步里程、路况、特色植物等

### CSV 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| id | 山的编号 | 1 |
| name | 山的名字 | 紫霞山 |
| area | 所在区县 | 龙泉驿 |
| distance | 距离成都的距离 | 50km |
| drive_time | 开车时长 | 1h 30min |
| difficulty | 星级评定 | ★ 到 ★★★★★ |
| scenery | 景点特色 | 俯瞰苍翠和龙泉驿市区 |
| plants | 当地植物和特产 | 青冈树、慈竹 |
| hiking_distance | 徒步距离 | 3.3km |
| road_condition | 路况描述 | 石梯、木栈道 |

## 🎯 使用流程

1. **点击"开始"** → 山名快速滚动切换
2. **点击"停止"** → 立即停止在某座山
3. **查看信息** → 显示该山的详细信息
   - 距离和车程
   - 徒步里程和路况
   - 难度等级
   - 风景评分
   - 植物特产
4. **做出决定**：
   - 满意 → 点击"就这个了！"保存到历史记录
   - 不满意 → 点击"再抽一次"重新开始

## 🎨 技术栈

- **Vue 3** - 渐进式前端框架

- **PapaParse** - CSV 解析库
- **LocalStorage** - 本地数据存储
